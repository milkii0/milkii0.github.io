前后端分离

reverse-sourcemap 下载地址:https://github.com/davidkevork/reverse-sourcemap

  1. 安装 nodejs

    apt install npm

  2. reverse-sourcemap

    npm install --global reverse-sourcemap

  3. 使用.js.map 文件获取前端源码

    reverse-sourcemap -v xxx.js.map -o output
